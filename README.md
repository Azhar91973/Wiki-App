 # Wiki-App
 

## Tech Stack
- **Language:** Kotlin  
- **Architecture:** MVVM (Model-View-ViewModel)  
- **Networking:** Retrofit + Coroutines  
- **Local Storage:** Room Database + Paging 3  
- **Dependency Injection:** Hilt  
- **Image Loading:** Glide  
- **Navigation:** Jetpack Navigation Component
